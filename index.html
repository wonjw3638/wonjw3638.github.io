Stereoscopic Web VR Project

Web 상에서 HMD 기기로 VR을 시청할 수 있는 Web VR을 제작했습니다. Stereoscopic을 구현하기 위해서 HMD기기 양안에 2개의 player를 각각 불러왔고, 양안시차가 존재하는 영상을 좌안, 우안에 각각 재생하여 더 몰입감 있는 VR영상을 제공합니다.

Interactive Web using CycleGAN

CycleGAN을 활용해 사용자가 업로드 한 이미지의 스타일을 변환해주는 모델을 학습 시켰습니다. 변환된 이미지는 화질개선을 한번 거친 후, 반응형 웹을 통해 결과 이미지를 제공합니다. 사용자는 Day <-> Night, Spring <-> Autumn, Summer <-> Winter 의 이미지 스타일 변환을 할 수 있습니다.

Tech Crawling Web page Project

Web Crawling을 사용하여 기업별 / 기술별로 게시글을 확인 할 수 있는 Web page를 제작 중에 있습니다. 사용자는 기업별 게시판에 접근하면 해당 기업에서 관심을 갖고 있는 기술들이 무엇인지 확인할 수 있으며 그에 관련된 게시글들을 한곳에 모아볼 수 있습니다. 기술별 게시판에 접근하면 기술별로 어느 기업에서 관심을 갖고 있는지 확인할 수 있습니다. 관심있는 게시글에는 북마크를 하여 후에 프로필에서 모아서 확인할 수 있습니다. 각 페이지에서는 사용자들끼리 의견을 주고 받을 수 있는 게시판을 구현할 예정입니다.
